# AI Web 2026 Project

This repository contains a simple static showcase page for an AI Web 2026 final project.

## Files

- `index.html`: Main landing page with a hero section, screenshot placeholder, and embedded markdown content.
- `style.css`: Page styling for layout, buttons, and markdown rendering.
- `contents.md`: Project summary content rendered inline using `zero-md`.
- `screenshot.png`: Demo screenshot placeholder used in the page.

## Run locally

Open `index.html` in your browser, or serve the folder with a static web server.

## Customize

- Update the hero text and metadata in `index.html`.
- Replace `screenshot.png` with your project screenshot.
- Edit `contents.md` with your project summary and details.

## Notes

This page uses `zero-md` to render `contents.md` inline in the browser.
